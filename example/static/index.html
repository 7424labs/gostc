gostc Static File Server

High-performance static file server with compression and caching

Features

  • Gzip & Brotli compression
  • In-memory cache with TTL
  • File watching with automatic cache invalidation
  • Rate limiting per IP
  • CORS support
  • Security headers
  • Prometheus metrics
  • HTTP/2 support

Test Files

  • JavaScript file (will be compressed)
  • CSS file (will be compressed)
  • JSON data (will be compressed)
  • SVG image (will be compressed)
